# YunTu-图书馆
本项目基于呼伦贝尔学院图书馆实际场景进行设计与开发，整合内蒙古20所高校图书馆资源及权威学术平台，实现了“找书高效、资源激活、座位有序、咨询即时、通知精准、体验创新”的全场景升级。

核心功能
模块：功能
🔍 智能找书： 馆藏检索、文献聚合、到书提醒
📚 智能资源服务： 主题书单、智能导读、个性化推荐
💺 智能座位管理： 热力图、预约系统、超时释放
🤖 AI智能咨询； 7×24问答、人工转接、知识库
📢 智能消息通知； 分类管理、精准推送、勿扰模式
🏫 高校直达； 整合20+高校及学术平台入口

技术栈
· 前端: React 18 + TypeScript + Vite + Tailwind CSS + ECharts
· 后端: Spring Boot 3 + Spring Cloud Alibaba + MySQL + Redis + ES
· AI: 通义千问/文心一言 + LangChain + BERT

快速开始
```bash
# 克隆项目
git clone https://github.com/yourname/yuntu-smart-library.git
# 安装依赖
cd yuntu-smart-library
npm install
# 启动开发服务器
npm run dev
```
